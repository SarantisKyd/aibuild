# AIBuild — Replit Setup Guide

## Files
- server.js    → Node.js backend (Express + Stripe + Replit DB)
- public/
  └── index.html → Frontend (paste index.html here)
- package.json

## Step 1 — Create Replit project
New Repl → Node.js template

## Step 2 — Add files
- Paste server.js content into server.js
- Create a folder called "public"
- Paste index.html into public/index.html
- Replace package.json content

## Step 3 — Install dependencies
In Replit Shell:
  npm install

## Step 4 — Add Stripe secrets
Click the 🔒 Secrets tab in Replit and add:

  STRIPE_SECRET_KEY       →  sk_test_...  (from Stripe Dashboard → API keys)
  STRIPE_PUBLISHABLE_KEY  →  pk_test_...
  STRIPE_WEBHOOK_SECRET   →  whsec_...   (see Step 5)
  APP_URL                 →  https://YOUR-REPL-NAME.YOUR-USERNAME.repl.co

## Step 5 — Set up Stripe webhook
1. Go to Stripe Dashboard → Developers → Webhooks
2. Add endpoint: https://YOUR-REPL.repl.co/webhook
3. Select event: checkout.session.completed
4. Copy the signing secret → paste as STRIPE_WEBHOOK_SECRET

## Step 6 — Run
Hit the Run button. Your app is live.

## How the money flows

  Client posts job ($150)
      ↓
  Clicks "Fund escrow" → Stripe Checkout
      ↓
  Stripe charges $150 to client card
  Webhook fires → job status = "funded"
  Job appears on live board
      ↓
  Builder bids and wins
  Builder does the work
  Builder marks as "delivered"
      ↓
  Client approves
  POST /api/jobs/:id/release
      ↓
  Stripe transfer → builder's Express account: $127.50
  Platform keeps: $22.50 (15%)
